# 🛡️ VulnSpecter

## Cyber Vulnerability Scanner

<img src="dashboard.png" width="900">

**VulnSpecter** is a lightweight cyber‑themed vulnerability scanner that
analyzes websites for common security weaknesses and visualizes results
through a **cyber security dashboard with an attack meter**.

Live Demo: https://huggingface.co/spaces/harshi1305/vulnspecter

------------------------------------------------------------------------

## 🚀 Features

-   Web vulnerability scanning
-   Open port detection
-   Security header analysis
-   Outdated server detection
-   Cyber security dashboard
-   Attack risk meter

------------------------------------------------------------------------

## ⚙️ Tech Stack

-   Python
-   Streamlit
-   Requests
-   Socket


------------------------------------------------------------------------

## ▶️ Run Locally

git clone https://github.com/yourusername/vulnspecter.git\
cd vulnspecter\
pip install -r requirements.txt\
streamlit run app.py
